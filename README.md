# Jogo do Número Secreto

Este é um jogo simples onde o usuário tenta adivinhar um número secreto gerado automaticamente entre 1 e 10. O objetivo é praticar lógica de programação e JavaScript de forma rápida e divertida.

## Tecnologias utilizadas
- HTML  
- CSS  
- JavaScript  

## Como jogar
Para jogar, basta acessar o link abaixo:

**https://jogo-iota-tawny-71.vercel.app/**

O jogo abrirá diretamente no navegador, sem necessidade de instalação.
